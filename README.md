# google_ext_test
# 🚀 AI Search Assistant (Chrome Extension)

Окремий повноцінний інтерфейс для пошуку та локального ШІ-помічника прямо у вашому браузері.

### 📥 Встановлення розширення

[![Встановити розширення](https://img.shields.io/badge/Chrome-Завантажити%20розширення-blue?style=for-the-badge&logo=google-chrome&logoColor=white)](https://github.com/Splinter1978/google_ext_test/archive/refs/heads/main.zip)

**Щоб запустити розширення на своєму комп'ютері:**

1. Натисніть на синю кнопку вище, щоб завантажити `ZIP-архів` проєкту, та розпакуйте його в окрему папку.
2. Відкрийте Google Chrome та перейдіть за адресою `chrome://extensions/`.
3. Увімкніть **Режим розробника** (Developer mode) у правому верхньому кутку.
4. Натисніть кнопку **Завантажити розпаковане розширення** (Load unpacked) ліворуч і оберіть розпаковану папку.

*Тепер іконка додатка з'явиться у вашій панелі розширень!*
